# Software y Licencias — IT Support

## Catálogo de software aprobado

La empresa mantiene un catálogo de software aprobado, disponible en el portal
interno de IT Support. Las aplicaciones del catálogo ya pasaron una revisión de
seguridad y de cumplimiento, y pueden instalarse de forma autogestionada sin
necesidad de aprobación adicional. El catálogo incluye herramientas de
productividad, comunicación, diseño y desarrollo según el rol.

## Solicitar software nuevo

Cuando un empleado necesita una herramienta que no está en el catálogo, abre un
ticket en el portal indicando la aplicación, el caso de uso y el costo estimado. La
solicitud pasa por una evaluación de seguridad y, si tiene costo, por la aprobación
del manager y de Finanzas. Este proceso evita riesgos de seguridad y compras
duplicadas.

No está permitido instalar software descargado de fuentes no oficiales ni usar
versiones pirateadas, ya que representan un riesgo de seguridad y un problema legal.

## Gestión de licencias

Las licencias de software se administran de forma centralizada. Cuando un empleado
necesita una licencia de una herramienta paga, IT Support verifica si hay licencias
disponibles en el pool de la empresa antes de comprar una nueva. Esto optimiza
costos y evita pagar por licencias sin usar.

Las licencias asignadas a un empleado se liberan automáticamente cuando deja la
empresa o cuando deja de usar la herramienta, para que puedan reasignarse.

## Suscripciones SaaS

Muchas herramientas son servicios en la nube (SaaS) con suscripción por usuario. La
empresa centraliza estas suscripciones para tener visibilidad del gasto y del uso.
Antes de contratar una nueva herramienta SaaS, se verifica que no exista ya una
solución equivalente aprobada, para evitar la proliferación de herramientas
redundantes.

## Software de desarrollo

Los equipos de ingeniería tienen un catálogo ampliado con entornos de desarrollo,
editores y librerías. El uso de software de código abierto está permitido siempre
que su licencia sea compatible con el uso comercial. En caso de duda sobre una
licencia open source, se consulta con el equipo de seguridad y con Legal.

## Actualizaciones y parches

Mantener el software actualizado es una responsabilidad de seguridad. Las
actualizaciones críticas se aplican automáticamente a través del sistema de gestión
de dispositivos. Para el resto del software, se notifica al empleado cuando hay una
actualización disponible y se espera que la instale en un plazo razonable.

## Inventario de software

IT Support mantiene un inventario del software instalado en cada equipo, con fines
de seguridad y de gestión de licencias. Este inventario permite detectar software no
autorizado y planificar las renovaciones de licencias. El inventario registra la
aplicación y la versión, no el contenido de los archivos del empleado.

## Preguntas frecuentes

¿Puedo instalar una extensión de navegador? Solo las aprobadas. Algunas extensiones
acceden a datos sensibles, por eso pasan por revisión.

¿Cuánto tarda la aprobación de software nuevo? Depende de la evaluación de
seguridad, pero suele resolverse en pocos días hábiles.

¿Puedo usar software libre? Sí, siempre que su licencia sea compatible con el uso
comercial. Ante la duda, consultá con IT Support.
