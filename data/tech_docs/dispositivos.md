# Gestión de Dispositivos — IT Support

## Notebook corporativa

Cada empleado recibe una notebook corporativa configurada por IT Support, con el
sistema operativo, las herramientas de seguridad y el software base ya instalados.
El equipo se envía al domicilio del empleado remoto antes de su primer día, o se
entrega en la oficina si trabaja de forma presencial. La notebook es propiedad de la
empresa y debe devolverse al finalizar el contrato.

Los empleados deben mantener el sistema operativo y las aplicaciones actualizados.
Las actualizaciones de seguridad críticas se instalan automáticamente, y se pide al
empleado reiniciar el equipo cuando se le solicite para completar la instalación.

## Gestión centralizada (MDM)

Todas las notebooks corporativas están inscriptas en un sistema de gestión de
dispositivos (MDM). El MDM permite a IT Support aplicar políticas de seguridad,
instalar software aprobado y, en caso de robo o pérdida, bloquear o borrar el equipo
de forma remota. El MDM no accede a los archivos personales del empleado; su función
es de seguridad y administración.

## Cifrado de disco

El disco de cada notebook corporativa está cifrado de forma obligatoria. Esto
protege la información en caso de robo o pérdida: sin las credenciales del empleado,
los datos no pueden leerse. El cifrado se activa automáticamente durante la
configuración inicial del equipo.

## Robo o pérdida de un dispositivo

Si una notebook corporativa es robada o se pierde, el empleado debe reportarlo a IT
Support de inmediato, idealmente dentro de la primera hora. IT Support bloquea el
acceso del equipo, revoca las sesiones activas y, si corresponde, borra el contenido
de forma remota. Reportar rápido es clave para proteger la información.

## Dispositivos móviles

Los empleados que acceden al correo corporativo o a herramientas internas desde su
teléfono deben inscribir el dispositivo en el sistema de gestión y aceptar las
políticas mínimas de seguridad: bloqueo de pantalla, cifrado y la posibilidad de
borrado remoto del contenido corporativo (no del personal). El uso del celular
personal para trabajo es opcional.

## Software permitido

Solo se permite instalar software aprobado por IT Support. El catálogo de
aplicaciones aprobadas está disponible en el portal interno y permite la instalación
autogestionada. Para instalar una herramienta que no esté en el catálogo, se abre un
ticket solicitando su evaluación, ya que todo software nuevo pasa por una revisión
de seguridad.

## Mantenimiento y reemplazo

Las notebooks se renuevan cada 3 años o antes si presentan fallas que afectan la
productividad. Cuando un equipo falla, el empleado abre un ticket y, según el caso,
IT Support repara el equipo o envía un reemplazo. Para empleados remotos, el
reemplazo se coordina por envío.

## Periféricos y accesorios

Los periféricos básicos (mouse, teclado, auriculares) se cubren con el estipendio de
home office. Para necesidades específicas vinculadas al rol o por motivos de
ergonomía y salud, el empleado puede solicitar accesorios adicionales mediante un
ticket con la justificación correspondiente.

## Preguntas frecuentes

¿Puedo instalar cualquier programa en mi notebook? Solo software aprobado. Para
otros, abrí un ticket para que IT Support lo evalúe.

¿Qué hago si pierdo mi notebook? Reportalo a IT Support de inmediato para que
bloqueen el acceso y protejan la información.

¿Cada cuánto se renueva el equipo? Cada 3 años, o antes si presenta fallas que
afectan tu trabajo.
